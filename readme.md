# TP2
## La nutrition
Une bonne nutrition repose sur un équilibre entre les différents nutriments essentiels dont le corps a besoin pour fonctionner de manière optimale. Ces nutriments se divisent en macronutriment — les glucides, les protéines et les lipides — et en micronutriments, comme les vitamines et les minéraux.
